# วิธีเริ่มใช้งานใหม่

<br><br>
**1.ติดตั้งตำสั่งผู้ใช้ใหม่**:<br>
```
pkg update -y
pkg upgrade -y
pkg install git
pkg install python
```
<br><br>
**2.ติดตั้งไฟล์**:<br>
```
git clone https://github.com/EH4404/freefc

cd freefc
```
<br><br>

**3.โค๊ดยิงเบอร์**:<br>
```
pip install -U -r requirements.txt

python main.py
```
